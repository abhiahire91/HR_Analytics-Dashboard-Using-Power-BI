<h1 align="center" style="color:#9b59b6;">📊 HR Analytics Dashboard</h1>

<p align="center">
  <b>🚀 Interactive HR Insights Dashboard built using Power BI</b><br>
  Transforming raw employee data into actionable business insights
</p>

<hr>

<h2>📌 Project Overview</h2>
<p>
This project is a <b>Power BI HR Analytics Dashboard</b> designed to analyze employee data and uncover key insights related to 
<b>attrition, salary distribution, job satisfaction, and workforce demographics</b>.
</p>

<ul>
  <li>📊 Helps HR teams understand employee turnover</li>
  <li>📉 Identifies key factors behind attrition</li>
  <li>📈 Provides data-driven decision support</li>
</ul>

<hr>

<h2>⚙️ Tools & Technologies</h2>

<ul>
  <li>📊 Power BI (Data Visualization)</li>
  <li>🧮 DAX (Data Analysis Expressions)</li>
  <li>📂 Data Cleaning & Transformation</li>
  <li>📁 Excel / CSV Dataset</li>
</ul>

<hr>

<h2>🚀 Key Features</h2>

<ul>
  <li>✔️ Employee Count & Attrition Overview</li>
  <li>✔️ Attrition Rate Analysis (16%)</li>
  <li>✔️ Salary Slab Distribution</li>
  <li>✔️ Attrition by Job Role</li>
  <li>✔️ Job Satisfaction Analysis</li>
  <li>✔️ Attrition by Age & Experience</li>
  <li>✔️ Gender-based Attrition Insights</li>
</ul>

<hr>

<h2>📊 Dashboard Insights</h2>

<ul>
  <li>🔹 High attrition observed in <b>Life Sciences & Medical</b> background</li>
  <li>🔹 Maximum employees fall under <b>5K salary slab</b></li>
  <li>🔹 Sales & Laboratory roles show higher attrition</li>
  <li>🔹 Early career employees (0–5 yrs) more likely to leave</li>
</ul>

<hr>

<h2>🎯 Business Impact</h2>

<ul>
  <li>📌 Improves employee retention strategies</li>
  <li>📌 Helps HR identify high-risk attrition groups</li>
  <li>📌 Enables data-driven workforce planning</li>
</ul>
