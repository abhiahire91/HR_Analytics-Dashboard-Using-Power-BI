📊 HR Analytics Dashboard
📌 Overview
This project presents an interactive HR Analytics Dashboard built using Power BI to analyze employee attrition and workforce trends.

🎯 Objective
Identify attrition patterns
Analyze key factors affecting employee turnover
Support data-driven HR decisions

🚀 Key Features
Employee count, attrition & attrition rate
Attrition by age, education, salary & job role
Job satisfaction analysis
Department-wise filtering

📊 Key Metrics
Employees: 961
Attrition: 133
Attrition Rate: 13.8%
Avg Age: 37 | Avg Salary: 6.3K

🛠️ Tools Used
Power BI
Excel / CSV

💡 Insights
Higher attrition in low salary range
Most churn in 26–35 age group
Research & Lab roles show high attrition
